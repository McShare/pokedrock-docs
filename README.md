# pokedrock-docs
SERP Pokédrock模组文档
<https://pokedrock.wiki/>

## 预览
```shell
yarn dev
```

## 构建
``` shell
yarn build
```

## 目录结构
```
└─ docs
   ├─ .vuepress
   |  ├─ configs
   |  |  └─ navbar # 顶栏配置文件
   |  |  └─ sidebar # 侧边栏配置文件
   |  ├─ public
   |  |  └─ images # 图片文件夹
   |  └─ config.js # 配置文件
   └─ languages # 其他语言文件夹
      └─ en-US # 英语
```
