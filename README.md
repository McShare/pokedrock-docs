# pokedrock-docs
SERP Pokédrock模组文档
<https://pokedrock.wiki/>

## 贡献
请通过 Fork/Pull Request 的方式帮助我们完善文档！

## 文档结构
```
└─ docs
   ├─ .vuepress
   |  ├─ configs
   |  |  └─ navbar # 顶栏配置文件
   |  |  └─ sidebar # 侧边栏配置文件
   |  ├─ public
   |  |  └─ images # 图片文件夹
   |  └─ config.js # 配置文件
   ├─ how-to-use #用法文件夹
   ├─ illustration #图例
   └─ languages # 其他语言文件夹
      └─ zh-CN # 中文
      |  └─ how-to-use #用法
      |  └─ illustration #图例
      └─ es-ES # 西班牙语
         └─ how-to-use #用法
         └─ illustration #图例
```

## 运行
### 预览
```shell
yarn dev
```

### 构建
``` shell
yarn build
```
